# Sales Bonus Dashboard

Bu layihədə *AdventureWorks SQL database* və əlavə olaraq *ProductBonus.csv* faylı əsasında satış və bonus analizləri aparılmışdır.  
Əlavə CSV faylı Python vasitəsilə SQL bazasına yüklənmiş, sonra isə məlumatlar Power BI ilə birləşdirilərək analiz edilmişdir.  

## Fayllar
- dashboard.pbix → Hazırlanmış Power BI dashboard faylı
- ProductBonus.csv → Əlavə məlumat faylı
- images/dashboard.png → Dashboard-un ekran görüntüsü
- Readme.md → Layihə haqqında izah

## Dashboard Haqqında
Dashboard satış göstəricilərini, bonus məlumatlarını və müxtəlif analitik vizualları əhatə edir.  
Əsas məqsəd – satış və bonus məlumatlarını bir yerdə analiz etmək və idarəetmədə qərarverməni asanlaşdırmaqdır.
